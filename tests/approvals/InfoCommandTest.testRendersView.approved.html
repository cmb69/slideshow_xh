Slideshow 1.4-dev

Synopsis

<?=slideshow('FOLDER', 'OPTIONS')?>
{{{slideshow('FOLDER', 'OPTIONS')}}}
FOLDER:
The path of a folder relative to CMSimple_XH's image folder.
OPTIONS:
Optional settings in form of a query string (option1=value1&option2=value2). The following options are available:
order
The order of the images: "fixed" (alphabetically ordered; start with first image), "sorted" (alphabetically sorted; start with randomly chosen image) or "random" (randomly ordered).
effect
The kind of the transition.
easing
The acceleration effect.
delay
The initial delay in ms until the slideshow starts.
pause
The duration of the pause between the transitions in ms.
duration
The duration of the transition effect in ms.

System check

PHP version ≥ 7.1.0

CMSimple_XH version ≥ 1.7.0

Plib_XH version ≥ 1.2

Folder './plugins/slideshow/config/' writable

Folder './plugins/slideshow/css/' writable

Folder './plugins/slideshow/languages/' writable
